# STM32 HAL库集成说明

## 重要提示

当前代码中的linter警告是**正常的**，因为STM32 HAL库还没有实际集成到项目中。

这些警告会在以下情况下消失：
1. 使用STM32CubeMX生成项目并集成HAL库后
2. 手动集成标准外设库后
3. 配置正确的包含路径后

## 集成步骤

### 步骤1: 获取HAL库

**方式1: 使用STM32CubeMX（推荐）**
```bash
1. 打开STM32CubeMX
2. 选择STM32F108T6
3. 配置外设
4. 生成代码
5. 复制Drivers目录到项目HAL目录
```

**方式2: 下载标准外设库**
```bash
1. 从ST官网下载STM32F10x标准外设库
2. 解压到项目HAL目录
```

### 步骤2: 配置编译选项

在 `config.h` 中：
```c
// 如果使用HAL库，取消注释：
#define USE_HAL_DRIVER
```

### 步骤3: 编译

```bash
make clean
make all
```
